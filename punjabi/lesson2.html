Lesson 2

ਤੁਸੀਂ ਕੀ ਕਰਦੇ ਹੋ? Tusi ki karde ho? — What do you do? (to a man)
ਤੁਸੀਂ ਕੀ ਕਰਦੀ ਹੋ? Tusi ki kardi ho? — What do you do? (to a woman)

ਮੈਂ ਜੰਗਲ ਵਿੱਚ ਦੌੜਦਾ ਹਾਂ। Main jangal vich daurda haan. — I run in the forest. (male)
ਮੈਂ ਜੰਗਲ ਵਿੱਚ ਦੌੜਦੀ ਹਾਂ। Main jangal vich daurdi haan. — I run in the forest. (female)

ਮੈਂ ਲਾਇਬ੍ਰੇਰੀ ਵਿੱਚ ਸੌਂਦਾ ਹਾਂ।  Main library vich saunda haan. — I sleep in the library. (male)
ਮੈਂ ਲਾਇਬ੍ਰੇਰੀ ਵਿੱਚ ਸੌਂਦੀ ਹਾਂ।  Main library vich saundi haan. — I sleep in the library. (female)

ਮੈਂ ਪੰਜਾਬੀ ਸਿੱਖਦਾ ਹਾਂ।  Main Punjabi sikhda haan. — I learn Punjabi. (male)
ਮੈਂ ਪੰਜਾਬੀ ਸਿੱਖਦੀ ਹਾਂ।  Main Punjabi sikhdi haan. — I learn Punjabi. (female)

ਮੈਂ ਕਿਤਾਬ ਪੜ੍ਹਦਾ ਹਾਂ।  Main kitaab parhda haan. — I read a book. (male)
ਮੈਂ ਕਿਤਾਬ ਪੜ੍ਹਦੀ ਹਾਂ।  Main kitaab parhdi haan. — I read a book. (female)

ਮੈਂ ਸੇਬ ਖਾਂਦਾ ਹਾਂ। Main seb khaanda haan. — I eat an apple. (male)
ਮੈਂ ਸੇਬ ਖਾਂਦੀ ਹਾਂ। Main seb khaandi haan. — I eat an apple. (female)
Vocabulary 
ਜੰਗਲ (jangal) — forest
ਲਾਇਬ੍ਰੇਰੀ (library) — library
ਪੰਜਾਬੀ (Punjabi) — Punjabi (language)
ਕਿਤਾਬ (kitaab) — book
ਸੇਬ (seb) — apple
ਵਿੱਚ (vich) - in
Sentences 
ਤੁਸੀਂ ਕੀ ਕਰਦੀ ਹੋ? 
ਮੈਂ ਜੰਗਲ ਵਿੱਚ ਦੌੜਦੀ ਹਾਂ। ਮੈਂ ਲਾਇਬ੍ਰੇਰੀ ਵਿੱਚ ਸੌਂਦੀ ਹਾਂ। ਮੈਂ ਪੰਜਾਬੀ ਸਿੱਖਦੀ ਹਾਂ। ਮੈਂ ਕਿਤਾਬ ਪੜ੍ਹਦੀ ਹਾਂ। ਮੈਂ ਸੇਬ ਖਾਂਦੀ ਹਾਂ।
        
ਤੁਸੀਂ ਕੀ ਕਰਦੇ ਹੋ? 
ਮੈਂ ਜੰਗਲ ਵਿੱਚ ਦੌੜਦਾ ਹਾਂ। ਮੈਂ ਲਾਇਬ੍ਰੇਰੀ ਵਿੱਚ ਸੌਂਦਾ ਹਾਂ। ਮੈਂ ਪੰਜਾਬੀ ਸਿੱਖਦਾ ਹਾਂ। ਮੈਂ ਕਿਤਾਬ ਪੜ੍ਹਦਾ ਹਾਂ। ਮੈਂ ਸੇਬ ਖਾਂਦਾ ਹਾਂ।

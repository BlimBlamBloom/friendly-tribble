Lesson 3

ਤੁਸੀਂ ਕੀ ਕਰਦੇ/ਕਰਦੀ ਹੋ? Tusi ki karde/kardi ho? — What do you do? (to a man/woman)

Run (ਦੌੜਨਾ / daudna) - ਮੈਂ ____ ਵਿੱਚ ਦੌੜਦਾ/ਦੌੜਦੀ ਹਾਂ।
ਮੈਂ ਜੰਗਲ ਵਿੱਚ ਦੌੜਦਾ/ਦੌੜਦੀ ਹਾਂ। Main jangal vich daurda/daurdi haan. — I run in the forest.
ਮੈਂ ਸ਼ਹਿਰ ਵਿੱਚ ਦੌੜਦਾ/ਦੌੜਦੀ ਹਾਂ। Main shehar vich daurda/daurdi haan. — I run in the city.
ਮੈਂ ਸਕੂਲ ਵਿੱਚ ਦੌੜਦਾ/ਦੌੜਦੀ ਹਾਂ। Main school vich daurda/daurdi haan. — I run in the school.
ਮੈਂ ਸਰੀ ਵਿੱਚ ਦੌੜਦਾ/ਦੌੜਦੀ ਹਾਂ। Main Sari vich daurda/daurdi haan. — I run in Surrey
ਮੈਂ ਲਾਇਬ੍ਰੇਰੀ ਵਿੱਚ ਦੌੜਦਾ/ਦੌੜਦੀ ਹਾਂ। Main library vich daurda/daurdi haan. — I run in the library.

Sleep (ਸੋਣਾ / sona) - ਮੈਂ ____ ਵਿੱਚ ਸੌਂਦਾ/ਸੌਂਦੀ ਹਾਂ।
ਮੈਂ ਲਾਇਬ੍ਰੇਰੀ ਵਿੱਚ ਸੌਂਦਾ/ਸੌਂਦੀ ਹਾਂ। Main library vich saunda/saundi haan. — I sleep in the library.
ਮੈਂ ਬੱਸ ਵਿੱਚ ਸੌਂਦਾ/ਸੌਂਦੀ ਹਾਂ। Main bus vich saunda/saundi haan. — I sleep in the bus.
ਮੈਂ ਸਕੂਲ ਵਿੱਚ ਸੌਂਦਾ/ਸੌਂਦੀ ਹਾਂ। Main school vich saunda/saundi haan. — I sleep in the school.
ਮੈਂ ਜੰਗਲ ਵਿੱਚ ਸੌਂਦਾ/ਸੌਂਦੀ ਹਾਂ। Main jangal vich saunda/saundi haan. — I sleep in the forest.
ਮੈਂ ਗੱਡੀ ਵਿੱਚ ਸੌਂਦਾ/ਸੌਂਦੀ ਹਾਂ। Main gadi vich saunda/saundi haan. — I sleep in the car.

Learn (ਸਿੱਖਣਾ / sikhna) – ਮੈਂ ____ ਸਿੱਖਦਾ/ਸਿੱਖਦੀ ਹਾਂ।
ਮੈਂ ਪੰਜਾਬੀ ਸਿੱਖਦਾ/ਸਿੱਖਦੀ ਹਾਂ। Main Punjabi sikhda/sikhdi haan. — I learn Punjabi.
ਮੈਂ ਅੰਗਰੇਜ਼ੀ ਸਿੱਖਦਾ/ਸਿੱਖਦੀ ਹਾਂ। Main angrezi sikhda/sikhdi haan. — I learn English.
ਮੈਂ ਮਿਊਜ਼ਿਕ ਸਿੱਖਦਾ/ਸਿੱਖਦੀ ਹਾਂ। Main music sikhda/sikhdi haan. — I learn music.
ਮੈਂ ਮੈਥ ਸਿੱਖਦਾ/ਸਿੱਖਦੀ ਹਾਂ। Main math sikhda/sikhdi haan. — I learn math.
ਮੈਂ ਸਾਇੰਸ ਸਿੱਖਦਾ/ਸਾਇੰਸ ਸਿੱਖਦੀ ਹਾਂ। Main science sikhda/sikhdi haan. — I learn science.

Read (ਪੜ੍ਹਨਾ / parhna) - ਮੈਂ ____ਪੜ੍ਹਦਾ/ਪੜ੍ਹਦੀ ਹਾਂ।
ਮੈਂ ਕਿਤਾਬ ਪੜ੍ਹਦਾ/ਪੜ੍ਹਦੀ ਹਾਂ। Main kitaab parhda/parhdi haan. — I read a book.
ਮੈਂ ਅਖਬਾਰ ਪੜ੍ਹਦਾ/ਪੜ੍ਹਦੀ ਹਾਂ। Main akhbaar parhda/parhdi haan. — I read a newspaper.
ਮੈਂ ਈਮੇਲ ਪੜ੍ਹਦਾ/ਪੜ੍ਹਦੀ ਹਾਂ। Main email parhda/parhdi haan. — I read an email.
ਮੈਂ ਬਲੌਗ ਪੜ੍ਹਦਾ/ਪੜ੍ਹਦੀ ਹਾਂ। Main blog parhda/parhdi haan. — I read a blog.
ਮੈਂ ਸੁਨੇਸ਼ ਪੜ੍ਹਦਾ/ਪੜ੍ਹਦੀ ਹਾਂ। Main suneesh parhda/parhdi haan. — I read a message.

Eat (ਖਾਣਾ / khaana) - ਮੈਂ ___ ਖਾਂਦਾ/ਖਾਂਦੀ ਹਾਂ।
ਮੈਂ ਸੇਬ ਖਾਂਦਾ/ਖਾਂਦੀ ਹਾਂ। Main seb khaanda/khaandi haan. — I eat an apple.
ਮੈਂ ਪੀਜ਼ਾ ਖਾਂਦਾ/ਖਾਂਦੀ ਹਾਂ। Main pizza khaanda/khaandi haan. — I eat pizza.
ਮੈਂ ਰੋਟੀ ਖਾਂਦਾ/ਖਾਂਦੀ ਹਾਂ। Main roti khaanda/khaandi haan. — I eat roti
ਮੈਂ ਆਈਸਕ੍ਰੀਮ ਖਾਂਦਾ/ਖਾਂਦੀ ਹਾਂ। Main ice cream khaanda/khaandi haan. — I eat ice cream.
ਮੈਂ ਗਾਜਰ ਖਾਂਦਾ/ਖਾਂਦੀ ਹਾਂ। Main gaajar khaanda/khaandi haan. — I eat a carrot.

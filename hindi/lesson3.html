Lesson 3

आप क्या करते / करती हैं? Aap kya karte/karti hain? — What do you do? 

Running (दौड़ना / daudna) - मैं ____ में दौड़ता/दौड़ती हूँ।
मैं जंगल में दौड़ता/दौड़ती हूँ। Main jangal mein daudta/daudti hoon. — I run in the forest. 
मैं शहर में दौड़ता/दौड़ती हूँ। Main shahar mein daudta/daudti hoon. — I run in the city. 
मैं स्कूल में दौड़ता/दौड़ती हूँ। Main school mein daudta/daudti hoon. — I run in the school. 
मैं सरे में दौड़ता/दौड़ती हूँ। Main Surrey mein daudta/daudti hoon. — I run in Surrey. 
मैं पुस्तकालय में दौड़ता/दौड़ती हूँ। Main pustakalay mein daudta/daudti hoon. — I run in the library. 

Sleeping (सोना / sona) - मैं ___ में सोता/सोती हूँ।
मैं पुस्तकालय में सोता/सोती हूँ। Main pustakalay mein sota/soti hoon. — I sleep in the library. 
मैं बस में सोता/सोती हूँ। Main bas mein sota/soti hoon. — I sleep in the bus. 
मैं स्कूल में सोता/सोती हूँ। Main school mein sota/soti hoon. — I sleep in the school. 
मैं जंगल में सोता/सोती हूँ। Main jangal mein sota/soti hoon. — I sleep in the forest. 
मैं कार में सोता/सोती हूँ। Main car mein sota/soti hoon. — I sleep in the car. 

Learning (सीखना / seekhna) - मैं ____ सीखता/सीखती हूँ।
मैं हिंदी सीखता/सीखती हूँ। Main Hindi seekhta/seekhti hoon. — I learn Hindi. 
मैं अंग्रेज़ी सीखता/सीखती हूँ। Main Angrezi seekhta/seekhti hoon. — I learn English.
मैं संगीत सीखता/सीखती हूँ। Main sangeet seekhta/seekhti hoon. — I learn music. 
मैं गणित सीखता/सीखती हूँ। Main ganit seekhta/seekhti hoon. — I learn math. 
मैं विज्ञान सीखता/सीखती हूँ। Main vigyan seekhta/seekhti hoon. — I learn science. 

Reading (पढ़ना / padhna) - मैं ____ पढ़ता/पढ़ती हूँ।
मैं किताब पढ़ता/पढ़ती हूँ। Main kitaab padhta/padhti hoon. — I read a book. 
मैं अखबार पढ़ता/पढ़ती हूँ। Main akhbaar padhta/padhti hoon. — I read a newspaper. 
मैं ईमेल पढ़ता/पढ़ती हूँ। Main email padhta/padhti hoon. — I read an email. 
मैं ब्लॉग पढ़ता/पढ़ती हूँ। Main blog padhta/padhti hoon. — I read a blog. 
मैं संदेश पढ़ता/पढ़ती हूँ। Main sandesh padh ta/padhti hoon. — I read a message. 


Eating (खाना / khaana) - मैं ___ खाता/खाती हूँ।
मैं सेब खाता/खाती हूँ। Main seb khata/khati hoon. — I eat an apple. 
मैं पिज़्ज़ा खाता/खाती हूँ। Main pizza khata/khati hoon. — I eat pizza. 
मैं रोटी खाता/खाती हूँ। Main roti khata/khati hoon. — I eat roti. 
मैं आइसक्रीम खाता/खाती हूँ। Main ice cream khata/khati hoon. — I eat ice cream. 
मैं गाजर खाता/खाती हूँ। Main gaajar khata/khati hoon. — I eat a carrot.

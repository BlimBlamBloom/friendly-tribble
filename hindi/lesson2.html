Lesson 2

आप क्या करते हैं? Aap kya karte hain? — What do you do? (to a man)
आप क्या करती हैं? Aap kya karti hain? — What do you do? (to a woman)
    
मैं जंगल में दौड़ता हूँ। Main jangal mein daudta hoon. — I run in the forest. (male)
मैं जंगल में दौड़ती हूँ। Main jangal mein daudti hoon. — I run in the forest. (female)
    
मैं पुस्तकालय में सोता हूँ। Main pustakalay mein sota hoon. — I sleep in the library. (male)
मैं पुस्तकालय में सोती हूँ। Main pustakalay mein soti hoon. — I sleep in the library. (female)
    
मैं हिंदी सीखता हूँ। Main Hindi seekhta hoon. — I learn Hindi. (male)
मैं हिंदी सीखती हूँ। Main Hindi seekhti hoon. — I learn Hindi. (female)
    
मैं किताब पढ़ता हूँ। Main kitaab padhta hoon. — I read a book. (male)
मैं किताब पढ़ती हूँ। Main kitaab padhti hoon. — I read a book. (female)
    
मैं सेब खाता हूँ। Main seb khaata hoon. — I eat an apple. (male)
मैं सेब खाती हूँ। Main seb khaati hoon. — I eat an apple. (female)
Vocabulary 
जंगल (jangal) — forest
पुस्तकालय (pustakalay) — library
हिंदी (Hindi) — Hindi (language)
किताब (kitaab) — book
सेब (seb) — apple
में (mein) — in
Sentences 
आप क्या करती हैं?
 मैं जंगल में दौड़ती हूँ। मैं पुस्तकालय में सोती हूँ। मैं हिंदी सीखती हूँ। मैं किताब पढ़ती हूँ। मैं सेब खाती हूँ।
आप क्या करते हैं?

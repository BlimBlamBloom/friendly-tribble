Ejercicio C: Ordenar las palabras（排列單字）

1. pizza / Yo / como / （我吃比薩。）

Yo como pizza.

2. comes / frutas / Tú / （你吃水果。）

Tú comes frutas.

3. agua / bebo / Yo / （我喝水。）

Yo bebo agua.

4. empanadas / comemos / Nosotros / （我們吃餡餅。）

Nosotros comemos empanadas.

5. come / pollo / Él / （他吃雞肉。）

Él come pollo.

6. empanadas / y / agua / Yo / como / （我吃餡餅並喝水。）

Yo como empanadas y agua.

7. frutas / y / leche / comes / Tú / （你吃水果並喝牛奶。）

Tú comes frutas y leche.

8. té / y / empanadas / bebe / Ella / （她喝茶並吃餡餅。）

Ella bebe té y empanadas.

9. arroz / y / pollo / comemos / Nosotros / （我們吃飯和雞肉。）

Nosotros comemos arroz y pollo.

10. pescado / y / fruta / comen / Ellos / （他們吃魚和水果。）

Ellos comen pescado y fruta.
